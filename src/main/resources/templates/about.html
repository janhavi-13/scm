About This Project

This project is developed as part of my master's program to provide a comprehensive contact management solution. The application includes features such as adding, updating, and deleting contacts, exporting data, and switching between dark and light modes.

Project Goals

  • To create a user-friendly contact management system.
  • To implement secure login and data handling practices.
  • To offer customization options through dark and light modes.
  • To provide easy data backup and restoration features.

Technology Used

This project utilizes a range of technologies to deliver a robust and efficient contact management system:

  • Frontend: HTML, CSS, JavaScript, Bootstrap, Tailwind CSS
  • Backend: Java Spring Boot
  • Database: MongoDB
  • Authentication: Google and GitHub OAuth
